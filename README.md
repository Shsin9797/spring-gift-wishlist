# spring-gift-wishlist
# 0단계 :  jdbc 연동 해서 CRUD 구현 완료
# 1단계 : 유효성 검사 및 예외처리

- [] 상품 이름 길이 유효성 검사 
    - [] 상품이름의 길이가 1~15인지 확인
    - [] 유효하지 않는 경우
        - [] 예외처리
        - [] http 상태 코드 전송 (400 Bad Request)
        - [] 응답 메세지 전송 

- [] 상품이름에 사용불가능한 특수문자가 있는지 유효성 검사 
    - [] 상품이름에 특수문자가 있는지 확인 
    - []  ()[] + - &  / _ 외의 특수문자가 잇는지 확인 
    - [] 유효하지 않는경우
        - [] 예외처리
        - [] http 상태 코드 전송 (400 Bad Request)
        - [] 응답 메세지 전송

- [] 상품이름에 "카카오" 가 포함되어 있는지 유효성 검사 
    - [] "카카오"라는 단어가 포함되어 있는 지 확인
    - [] 유효하지 않는경우
        - [] 예외처리
        - [] http 상태 코드 전송 (400 Bad Request)
        - [] 응답 메세지 전송
